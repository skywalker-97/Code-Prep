\# Code Prep – Online Examination Platform (MERN Stack)



Code Prep is a full-stack MERN-based online examination platform where users can

register, log in, attempt MCQ exams, and view their results. Admin users can add

questions through a protected admin panel.



---



\## 🚀 Features

\- User Registration \& Login (JWT Authentication)

\- Role-based access (Admin / Student)

\- Protected Routes

\- Admin Panel to add MCQ questions

\- Online exam with auto evaluation

\- Result history dashboard

\- RESTful APIs

\- Responsive UI using React \& Tailwind CSS
